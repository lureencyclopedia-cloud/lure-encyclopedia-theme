# Lure Encyclopedia Theme

A custom Ghost theme built exclusively for Lure Encyclopedia.

## Mission

Lure Encyclopedia is a long-term editorial publication documenting the history, evolution, and practical use of fishing lures.

This theme is designed to reflect that mission through timeless typography, thoughtful layouts, and fast, accessible performance.

## Goals

- Editorial-first design
- Encyclopedia-quality presentation
- Excellent readability
- Fast loading
- Accessible
- Maintainable
- Mobile-first
- Built specifically for Ghost(Pro)
